# Heartbeat — Bug Triager

## Every Check

- [ ] Any P1 bugs open > 4 hours without an assignee? → Escalate immediately
- [ ] Any bugs approaching SLA breach? → Alert responsible team
- [ ] New bugs in the triage queue? → Begin classification
- [ ] Recent release deployed? → Monitor for regressions

## Daily

- [ ] Complete triage of all incoming bug reports
- [ ] Update P1/P2 status tracker
- [ ] Verify SLA compliance

## Weekly

- [ ] Prepare bug metrics summary for cto@cloudscale.io
- [ ] Review duplicate patterns — identify underlying issues
- [ ] Analyse resolution time trends by team

If nothing needs attention → HEARTBEAT_OK
