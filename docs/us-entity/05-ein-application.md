# EIN Application Process

## What is an EIN?

An Employer Identification Number (EIN) is the US equivalent of a company tax ID. Required for:
- Opening a US bank account
- Filing US tax returns
- Hiring employees/contractors
- Setting up payment processors (Stripe, etc.)

## Option A: Via Formation Service (Easiest)

If using **Firstbase** or **Stripe Atlas**, EIN application is included. They handle it for you.
- Timeline: 4–8 weeks for non-US applicants
- No additional cost

## Option B: DIY (Non-US Applicants)

Non-US residents **cannot** use the online IRS application (requires SSN/ITIN). You must apply by phone or fax.

### By Phone (Fastest DIY Method)

1. **Complete IRS Form SS-4** ([download](https://www.irs.gov/pub/irs-pdf/fss4.pdf))
2. Fill in:
   - **Line 1:** AfrexAI LLC
   - **Line 2:** (leave blank — no trade name)
   - **Line 3:** Christina Beckford (responsible party)
   - **Line 4a-b:** NYC virtual office address
   - **Line 7a:** Christina Beckford name
   - **Line 7b:** Her ITIN or SSN (write "Foreign" if none)
   - **Line 8a:** LLC
   - **Line 8b:** 2 (number of members)
   - **Line 9a:** Check "Banking purposes" and "Other" → write "Started new business"
   - **Line 10:** Reason: "Started new business"
   - **Line 11:** Date business started (formation date)
   - **Line 12:** First month of tax year → "January"
   - **Line 16:** Check appropriate NAICS code (541511 — Custom Computer Programming Services)
   - **Line 18:** Check "Yes" if this is the first EIN request
3. **Call IRS:** +1 (267) 941-1099 (International applicants line)
   - Hours: Mon–Fri, 6:00 AM – 11:00 PM EST
   - **Best time to call:** 6:00–7:00 AM EST (shortest wait)
   - Have SS-4 form in front of you
   - They'll issue EIN verbally on the call
   - Confirmation letter (147C) mailed to your address in 4–6 weeks

### By Fax

1. Complete Form SS-4 as above
2. Fax to: +1 (855) 641-6935
3. Receive EIN by fax within 4–5 business days (include fax number on form)

### By Mail (Slowest)

1. Complete Form SS-4
2. Mail to:
   ```
   Internal Revenue Service
   Attn: EIN International Operation
   Cincinnati, OH 45999
   ```
3. Receive EIN in 4–6 weeks

## After Receiving EIN

- [ ] Save EIN confirmation letter (147C) securely
- [ ] Update Operating Agreement with EIN
- [ ] Use EIN to open Mercury bank account
- [ ] Register for any state tax obligations
- [ ] File initial BOI report with FinCEN

## Important Notes

- Each LLC gets **one** EIN — it doesn't change
- EIN is public information (not secret like SSN)
- If you lose it, call IRS to retrieve
- **Do not apply more than once** — duplicate EINs cause IRS problems
