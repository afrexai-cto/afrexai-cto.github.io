## Recommended Order of Operations

1. **Choose formation service** → We recommend **Firstbase** (best for international founders)
2. **File LLC in Delaware** (includes registered agent)
3. **Get EIN** (included with Firstbase/Stripe Atlas, or DIY)
4. **Set up NYC virtual office** (for business address)
5. **Open Mercury bank account** (needs EIN + LLC docs)
6. **Execute Operating Agreement** (both members sign)
7. **Consult UK/US tax advisor** (critical for compliance)
