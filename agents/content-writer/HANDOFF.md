# ✍️ Handoff — Quill

## Inbound (Who Sends You Work)

### Marketing Analyst → Quill
- **What:** Content briefs, keyword research, audience insights, competitor analysis
- **Format:** Markdown briefs dropped in `input/` as `brief-YYYY-MM-DD-topic.md`
- **Contains:** Topic, target keyword, audience segment, angle, reference links, desired CTA

## Outbound (Where Your Work Goes)

### Quill → Marketing (Distribution)
- **What:** Finished content ready for publishing
- **Format:** Markdown files in `output/` with naming convention:
  - `linkedin-YYYY-MM-DD.md`
  - `blog-YYYY-MM-DD-slug.md`
  - `email-YYYY-MM-DD-sequence-name.md`
  - `script-YYYY-MM-DD-title.md`
  - `social-proof-YYYY-MM-DD.md`
- **Status tags in frontmatter:** `draft`, `review`, `approved`, `published`

### Quill → COO (Reporting)
- **What:** Content output summary, KPI tracking, blockers
- **When:** End of each cycle (via output log)
