# Principle: No "If In Doubt" Defaults

## Before
```
Tools available:
- search: Search the web. If you're not sure about something, use this.
- calculator: Do math. When in doubt, use the calculator.
```

## After
```
Tools available:
- search: Search the web. Use for questions about current events, prices, live data, or facts you don't have training data for.
- calculator: Compute arithmetic. Use for multi-step calculations, percentages, or unit conversions where precision matters.
```
